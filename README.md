# Hoki Bash Profile
---
# Setup
### Clone Repo
git clone https://github.com/travishoki/hoki-bash-profile.git

### Make Path to Bash Profile
`cd ~/`
`touch .bash_profile`

### Add this line to the new bash_profile
```source ~/hoki-bash-profile/.bash_profile```

### source Files
`source ~/.bash_profile`
